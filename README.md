# 🛡️ SentryFlow Console

A real-time monitoring dashboard built with React.js that helps developers track, analyze, and manage application logs efficiently. The platform provides live log streaming, advanced filtering, custom alert management, server health monitoring, regex-based searching, and real-time error frequency visualization through an intuitive and responsive interface.

SentryFlow Console was designed to address the challenge of monitoring large volumes of application logs while maintaining high performance and usability. By leveraging virtualization, Redux Toolkit state management, and optimized React rendering techniques, the dashboard remains responsive even when processing thousands of log entries.

## Setup Instructions

```bash
git clone https://github.com/sanjana123642/sentryflow-console.git

cd sentryflow-console

npm install --legacy-peer-deps

npm run dev
```

Application URL:

```text
http://localhost:5173
```

## Production Build

```bash
npm run build
npm run preview
```
